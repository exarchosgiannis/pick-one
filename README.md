# **PickOne**  
https://pickonex.vercel.app/

Clone of Chooser app.
A tiny project where everyone puts a finger on the screen and the app randomly picks one person. 
Super simple, nothing fancy. I originally used Chooser app from the App Store, but it recently became paid, and I didn’t want to pay for it.

So I made my own version and put it here in case someone else wants it too :)

You can also add it to your Home Screen (iPhone & Android) and use it like a normal app:
- iPhone → Safari → Share → Add to Home Screen  
- Android → Chrome → Menu → Add to Home Screen
